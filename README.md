![运行命令](./__assets/运行命令.png)

分别是使用g++编译器编译生成Unix可执行文件SyntaxAnalysis，并运行该可执行文件。运行该可执行文件时，后面需要附上所要执行的文件名称作为**main**函数的参数，才可对所选文件进行语法分析，如果没有参数，则程序会运行错误。
